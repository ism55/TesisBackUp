Se logró diseñar un prototipo de sistema de acceso remoto al controlador del brazo manipulador MA2000 ubicado en el LCID.

El brazo robot puede manejarse a través del software Robocom desarrollado en \cite{juan1} comunicándose a través de un convertidor USB-serial conectado al puerto serial de un microcontrolador Microchip dsPIC30F3011. El prototipo realizado en este proyecto agregó una mejora en el manejo del brazo robot, permitiendo que este pueda utilizarse sin el software, accediendo a una página web sin usar el módulo convertidor USB ? Serial FTDI232.

La utilización del microcontrolador ESP8266 para establecer comunicación vía Wi-Fi a través de un servidor alojado en el sistema embebido, demostró ajustarse a las necesidades del proyecto. Este microcontrolador ayudó a establecer una comunicación en tiempo real en la generación de movimientos.
