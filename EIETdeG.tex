\textbf{Ismael J. Bautista G.}

\begin{large}
\textbf{DISEÑO DE UN PROTOTIPO DE SISTEMA DE ACCESO REMOTO AL CONTROLADOR DE UN BRAZO MANIPULADOR}
\end{large}
\end{center}

\noindent%
\textbf{Tutor Académico: Tamara Pérez. Tesis.
Caracas, Universidad Central de Venezuela. Facultad de Ingeniería.
Escuela de Ingeniería Eléctrica. Mención Electrónica Computación y Control. Año 2019, 123 pp.}

\noindent
\textbf{Palabras Claves:} Brazo Manipulador, LCID, Acceso remoto, Wi-Fi, Servidor web, Android, Robótica. \\[1ex]

\noindent \textbf{Resumen.-} 

El principal objetivo de este trabajo fue el diseño de un prototipo de sistema de acceso remoto al controlador del brazo manipulador MA2000. Partiendo de un software computarizado realizado en un trabajo anterior y el deshuso del equipo, surgió la necesidad de una mejora del sistema original que permitió utilizarlo sin necesidad de computadoras, a través de un sistema remoto que realizó la adaptación. Para el diseño de este prototipo se estudiaron varias dispositivos compatibles con tecnología remota donde finalmente se seleccionó el microcontrolador ESP8266 como hardware para implementar el prototipo, debido a que contó con los requerimientos básicos necesarios para el alcance propuesto de este sistema. Se diseñó una interfaz web alojada en un servidor dentro del microcontrolador el cuál se puede acceder a través de una red inalámbrica vía Wi-Fi. Como soporte a la interfaz principal, se diseñó una aplicación en Android Studio que permitió la lectura de archivos de texto para la programación de rutinas de movimiento. La solución propuesta al problema inicial se logró con los objetivos planteados, pudiendo manejar el brazo manipulador sin presencia del software original ni conexiones alámbricas diferentes al puerto serial del microcontrolador. Como documento de respaldo para la utilización, se generó un manual en línea en el que se encuentran las características de configuración y uso del prototipo, así como material audiovisual del funcionamiento de este.
\end{spacing}
